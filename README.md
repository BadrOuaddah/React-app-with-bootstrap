### Prerequisites

* npm
  ```bash
  npm install npm@latest -g
  ```

* bootstrap
  ```bash
  npm install bootstrap -g
  ```

* react-bootstrap
  ```bash
  npm install react-bootstrap
  ```

### Installation

+ Clone the repo
   
   ```bash
   git clone https://github.com/BadrOuaddah/React-app-with-bootstrap.git
   ```
+ Run the app
   
   ```bash
   npm start
   ```
